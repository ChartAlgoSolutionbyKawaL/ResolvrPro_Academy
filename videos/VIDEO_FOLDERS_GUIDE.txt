HOW TO ADD VIDEOS:
------------------

1. PLACE VIDEO FILES:
   Copy your .mp4 or .webm video files into the correct session folder.
   Example: videos/explorer/session-01/video-1.mp4

2. EDIT videos.json:
   Each session folder has a videos.json template. Edit it to match
   your actual video files:

   {
     "level": "explorer",
     "session": 1,
     "videos": [
       {
         "title": "Introduction to Future Skills",
         "description": "What are future skills and why they matter",
         "file": "video-1.mp4",
         "duration": "5:30",
         "type": "lecture"
       },
       {
         "title": "Growth Mindset Explained",
         "description": "Fixed vs growth mindset with real examples",
         "file": "video-2.mp4",
         "duration": "8:15",
         "type": "demo"
       }
     ]
   }

3. VIDEO TYPES (for gallery labels):
   - "lecture"    — Teaching/explanation video
   - "demo"       — Practical demonstration
   - "summary"    — Recap and key takeaways
   - "activity"   — Guided activity walkthrough
   - "interview"  — Expert or peer interview
   - "review"     — Homework/project review

4. OPTIONAL — THUMBNAILS:
   Add a thumbnail image (jpg/png) and reference it in videos.json:
   {
     "title": "My Video",
     "file": "video-1.mp4",
     "thumbnail": "thumb-1.jpg",
     ...
   }

5. OPTIONAL — YOUTUBE/VIMEO EMBEDS:
   Instead of local files, you can use YouTube or Vimeo URLs:
   {
     "title": "My Video",
     "file": "https://www.youtube.com/embed/VIDEO_ID",
     "duration": "5:30",
     "type": "lecture"
   }
