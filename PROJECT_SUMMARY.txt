YOUR NEXT STEPS:
----------------

1. DOWNLOAD THE VIDEOS
   Download or create videos for each session topic.
   Recommended: 3-5 videos per session (5-10 min each)
   Types: Lecture, Demo, Summary, Activity, Interview

2. PLACE VIDEOS IN FOLDERS
   Copy your .mp4 files to the correct session folder:
   Example: videos/explorer/session-01/video-1.mp4

3. EDIT videos.json FILES
   Each session folder has a videos.json template.
   Update it to match your actual video files:
   
   {
     "level": "explorer",
     "session": 1,
     "videos": [
       {
         "title": "Introduction to Future Skills",
         "description": "What are future skills and why they matter",
         "file": "video-1.mp4",
         "duration": "5:30",
         "type": "lecture"
       }
     ]
   }

4. OPTIONAL: ADD THUMBNAILS
   Create thumbnail images (jpg/png) for each video
   and reference them in videos.json:
   "thumbnail": "thumb-1.jpg"

5. OPTIONAL: USE YOUTUBE/VIMEO
   Instead of local files, use embed URLs:
   "file": "https://www.youtube.com/embed/VIDEO_ID"

6. TEST THE SITE
   Open index.html in a browser
   Click any session in LMS or Curriculum
   Click the "🎬 Videos" tab
   Videos should appear as cards in the gallery
   Click a card to open the video player
